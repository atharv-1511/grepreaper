grepreaper Package

Efficient Pattern Matching and Data Filtering for R

The grepreaper package provides efficient file reading capabilities using the power of grep at the command line. This implementation significantly improves performance when working with large text datasets in R.

Core Functions

1. Pattern Counting with grep_count()

library(grepreaper)

# Count occurrences of "IT" in employee data
grep_count("employee_data.csv", "IT")
[1] 15

2. Reading Filtered Data with grep_read()

# Read only employees in IT department
it_employees <- grep_read("employee_data.csv", "IT")
head(it_employees)

3. Multiple File Support

# Count matches across multiple files
grep_count(c("2021.csv", "2022.csv", "2023.csv"), "IT")

Function Implementation

grep_count Function

grep_count <- function(files, pattern, invert = FALSE, ignore_case = FALSE,
                fixed = FALSE, show_cmd = FALSE, recursive = FALSE,
                word_match = FALSE) {
  
  # Input validation
  if (!is.character(files) || length(files) == 0) {
    stop("'files' must be a non-empty character vector")
  }
  
  # Build grep options
  options <- ""
  if (invert) options <- paste(options, "-v")
  if (ignore_case) options <- paste(options, "-i")
  if (fixed) options <- paste(options, "-F")
  if (recursive) options <- paste(options, "-r")
  if (word_match) options <- paste(options, "-w")
  
  # Build the command
  cmd <- build_grep_cmd(pattern, files, options, count = TRUE)
  
  # Return command if requested
  if (show_cmd) {
    return(cmd)
  }
  
  # Execute command and process results
  result <- safe_system_call(cmd)
  
  # Process results...
}

Common Parameters

invert: Return non-matching rather than matching lines

ignore_case: Perform case-insensitive matching

fixed: Treat pattern as a fixed string, not a regular expression

recursive: Search directories recursively

word_match: Match only whole words

show_cmd: Display the command being executed
